# RAWGVideoGames

A simple list/detail app that follows <i> some </i> recommendations laid out in the <a href="https://developer.android.com/jetpack/guide">Guide to app architecture</a>

## Concepts 

  * Kotlin
  * Jetpack compose
  * Coroutines 
  * Retrofit
  * okHttp
  * Hilt
  * Coil
  * Compose Navigation
  * DataStore
  * Horizontial Pager
  * FlowRow
